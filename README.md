## Building

Clone the repository:

```bash
  git clone https://github.com/or4acle/FutabaWare.git
```
Windows Powershell or IDEs:

```bash
  .\gradlew build
```

Linux/Mac:

```bash
  ./gradlew build
```
## Requirements

- Java 8: Necessary to run Minecraft 1.12.2.
- Minecraft Forge: 1.12.2 - 14.23.5.2768.
